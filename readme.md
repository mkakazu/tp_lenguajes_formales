# Trabajo practico para la materia Lenguajes Formales, FIUBA


Enunciado: completar un interprete del lenguaje de programacion Applesoft Basic
implementado en Clojure

Instrucciones de uso:

en consola, ejecutar un interprete de clojure.
```
# por ejemplo
java -jar clojure-1.8.0.jar
```

cargar el codigo para interpretar applesoft basic
```
(load-file "basic.clj")
```

ejecutar el interprete de applesoft basic
```
(driver-loop)
```
